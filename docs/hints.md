# HTTP Body Types

## Text-Based Content Types

- `text/plain`
- `text/html`
- `text/css`
- `text/javascript`
- `application/javascript`
- `text/csv`

## Application-Based Content Types

- `application/json`
- `application/xml`
- `text/xml`

## Form Data Types

- `application/x-www-form-urlencoded`
- `multipart/form-data`

## Binary and File-Based Content Types

- `application/octet-stream`
- `application/pdf`
- `image/png`
- `image/jpeg`
- `image/gif`
- `audio/mpeg`
- `audio/ogg`
- `video/mp4`
- `video/webm`

## Compressed Content Types

- `application/zip`
- `application/gzip`
- `application/x-tar`

## Application-Specific Content Types

- `application/vnd.api+json`
- `application/ld+json`
- `application/x-ndjson`
- `application/graphql`

## Other Content Types

- `application/x-msgpack`
- `application/x-protobuf`
